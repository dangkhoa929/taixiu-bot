Bot tự động dự đoán Tài/Xỉu Sunwin bằng AI (Markov, Cầu bệt, Voting...).

- Dự đoán thông minh Tài/Xỉu + xác suất
- Lưu log đúng/sai, xem thống kê với /history
- Giao diện Telegram trực quan
